# pulpfiction
Easy to use Android video and audio player build on ExoPlayer

## features
☐ support for DASH, HLS, MP4, MP3\
☐ play/pause/loading views\
☐ image for idle mode\
☐ info and error events\
☐ fullscreen mode\
☐ picture in picture mode\
☐ chromecast support\
☐ quality selection\
☐ share\
☐ error handlers\
☐ reconnect behaviors\
☐ stream downloaders\
☐ clip image\
☐ next/prev clip\
☐ ads support
